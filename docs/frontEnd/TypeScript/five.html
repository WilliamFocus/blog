五、TS枚举（Enumerate）

WilliamFocus 2021-05-05 TypeScript

——缺省值的枚举会转成属性名和属性值两两 (数值 0 开始递增、枚举成员名) 对应的映射关系的数据类型

// TS             // JS                                       // 转换结果
enum Day {        var Day;                                    { "0": "SUNDAY",
    SUNDAY,       (function (Day) {                             "1": "MONDAY",
    MONDAY,         Day[Day["SUNDAY"] = 0] = "SUNDAY";          "2": "TUESDAY",
    TUESDAY,        Day[Day["MONDAY"] = 1] = "MONDAY";          "3": "WEDNESDAY",
    WEDNESDAY,      Day[Day["TUESDAY"] = 2] = "TUESDAY";        "4": "THURSDAY",
    THURSDAY,       Day[Day["WEDNESDAY"] = 3] = "WEDNESDAY";    "5": "FRIDAY",
    FRIDAY,         Day[Day["THURSDAY"] = 4] = "THURSDAY";      "6": "SATURDAY",
    SATURDAY        Day[Day["FRIDAY"] = 5] = "FRIDAY";          "SUNDAY": 0,
  }                 Day[Day["SATURDAY"] = 6] = "SATURDAY";      "MONDAY": 1,
                  })(Day || (Day = {}));                        "TUESDAY": 2,
															    "WEDNESDAY": 3,
															    "THURSDAY": 4,
                                                                "FRIDAY": 5,
																															  "SATURDAY": 6 }

  • 数字枚举： 可以给任意位置枚举类型的成员一个赋初始数值，其后每个成员递增 +1；
  • 字符串枚举： 枚举成员的值就是我们显式指定的字符串字面量；
  • 异构枚举： 同时拥有数字和字符类型的成员；
  • 常量枚举 ......
  • 外部枚举 ......

💡 注意：

  • 定义枚举类型可以兼具值和类型于一体；
  • 枚举成员全部是字面量枚举值，则所有枚举成员既是值又是类型；
  • 若仅有一个成员且是字面量成员，那么这个成员的类型等于枚举类型，如类型Day = Day.MONDAY；
  • 数字枚举和数字类型相互兼容，数字枚举不稳定(可变)；
  • 两个枚举的值在 js 中即使恒等(===)，但 TS 中两个枚举值恒不相等；枚举仅和自身兼容；
